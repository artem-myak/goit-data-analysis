# Advanced SaaS Revenue & Cohort Analysis

## Project Overview
Building upon previous revenue data, this project implements advanced SaaS metrics to track business health and user retention through financial cohorts.

## Advanced BI Techniques:
* **New MRR Tracking:** Isolated revenue from new users in their acquisition month.
* **Growth Analytics:** Implemented Table Calculations to visualize Month-over-Month (MoM) revenue growth % alongside total figures.
* **Cohort Analysis:** * Developed a retention matrix showing revenue decay over time.
    * Applied custom gradient formatting based on the "Retention Index" (Ratio of current month revenue to the cohort's first month).
* **LOD Expressions:** Used Fixed Level of Detail expressions to identify each user's first payment date.
