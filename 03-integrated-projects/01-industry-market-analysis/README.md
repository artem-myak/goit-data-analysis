# Integrated Data Analysis Project: Industry Market Insights

## Project Overview
This is a comprehensive data analysis project that integrates Google Sheets for EDA/Visualization and SQL for complex data retrieval. The project follows the full analytical workflow: data cleaning, exploratory data analysis (EDA), business metric calculation via SQL, and dashboarding.

## Key Workflow Stages:
1. **Data Preparation & EDA:** * Cleaned raw datasets (handled duplicates, missing values, and formatting).
    * Conducted Exploratory Data Analysis to identify initial patterns and anomalies.
2. **SQL Analytics:**
    * Developed complex queries using `JOINs`, `Subqueries`, and `Aggregate functions`.
    * Calculated key business metrics like average order value (AOV), conversion rates, and sales dynamics.
3. **Visualization & Dashboarding:**
    * Created an interactive dashboard in Google Sheets.
    * Visualized KPIs and trends using line charts, bar graphs, and pivot tables.
4. **Insights & Recommendations:**
    * Summarized findings into actionable business conclusions.

## Technical Skills Applied:
* **Advanced Google Sheets:** Data cleaning, Pivot Tables, Conditional Formatting, Charting.
* **SQL (BigQuery/PostgreSQL):** Relational logic, data grouping, and multi-level aggregations.
* **Business Analysis:** KPI definition and data-driven storytelling.
