# A/B Test Planning: Subscription Screen Optimization

## Project Overview
This project focuses on the strategic planning of an A/B test for a mobile application's subscription screen. The goal is to evaluate the impact of "Anchoring Price" psychology (displaying a 50% discount) on the overall conversion rate from installation to purchase.

## Experiment Design:
* **Hypothesis:** Adding a "50% Discount" label to the $4.99 subscription price will increase conversion rates by leveraging perceived value.
* **Randomization Unit:** User Install (Full-funnel impact assessment).
* **Target Audience:** 100% of new users, split 50/50 between Control and Test groups.

## Metrics Framework:
* **Primary Metric:** Conversion Rate (Install to Purchase).
* **Guardrail Metrics:** Onboarding completion rate, ARPU (Average Revenue Per User), and Refund Rate.

## Statistical Parameters & Sample Size:
* **Baseline Conversion:** 5.78% (Total funnel conversion).
* **Minimum Detectable Effect (MDE):** 10% relative lift.
* **Confidence Level:** 95% ($\alpha = 0.05$).
* **Statistical Power:** 80% ($1 - \beta$).
* **Required Sample Size:** ~29,000 users (~14.5 days of testing).

## Strategic Outcomes:
Defined clear success criteria and action plans for both positive and negative results, including alternative testing ideas like "Free Trial" models.
