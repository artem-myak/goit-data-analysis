# A/B Test Results Analysis: Price Anchoring Effectiveness

## Project Overview
This project evaluates the results of an A/B test conducted on a mobile subscription screen. The goal was to determine if adding a "50% Discount" label (Price Anchoring) significantly impacts the conversion rate from installation to purchase.

## Analysis Workflow:
* **Exploratory Data Analysis:** Calculated group sizes, conversion counts, and baseline performance.
* **Statistical Hypothesis Testing:** Applied the **Z-test for Proportions** to determine the statistical significance of the observed lift.
* **Data Visualization:** Built comparison bar charts with 95% Confidence Intervals (CI) and time-series conversion plots.
* **Decision Making:** Formulated business recommendations based on p-value and confidence interval analysis.

## Key Technical Skills:
* **Statistics:** Hypothesis testing (H0 vs H1), p-value interpretation, Confidence Intervals.
* **Python Libraries:** Pandas, Statsmodels (Proportions Z-test), Seaborn, Matplotlib.
* **Product Sense:** Conversion Rate (CR) analysis and data-driven storytelling.
