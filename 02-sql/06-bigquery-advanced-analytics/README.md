# Advanced Data Analytics with Google BigQuery

## Project Overview
This project demonstrates advanced SQL techniques within the Google BigQuery environment. It focuses on handling complex data structures (NESTED & REPEATED fields), optimizing query performance using partitioning, and automating analytics through Scheduled Queries.

## Key Technical Skills:
* **Complex Data Types:** Managed nested data using `STRUCT` and collections using `ARRAY_AGG`.
* **Data Flattening:** Used `UNNEST()` to transform arrays back into a relational format.
* **Storage Optimization:** Leveraged `_PARTITIONDATE` to filter data, significantly reducing query costs.
* **Workflow Automation:** Configured **Scheduled Queries** to automate daily reporting.
* **Query Performance:** Evaluated data processing volume (MB/GB) for cost-efficiency.

## Deliverables
* `queries.sql`: Clean SQL code for all tasks (Arrays, Structs, Unnest).
* `BigQuery-Technical-Report.pdf`: A detailed document containing execution screenshots, scheduled query configurations, and data processing estimates.
