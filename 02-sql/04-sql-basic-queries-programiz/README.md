# Basic SQL Queries Analysis (Programiz Environment)

## Project Overview
This project demonstrates foundational SQL skills using the Programiz SQL Editor environment. It covers a wide range of operations including data filtering, string manipulation, conditional logic, and relational data joins across `Customers`, `Orders`, and `Shippings` tables.

## Key SQL Skills Applied:
* **Filtering & Sorting:** Used `WHERE`, `LIKE`, and `ORDER BY` to extract and organize specific data points.
* **String Functions:** Applied `CONCAT` (via `||`), `TRIM`, `LENGTH`, `UPPER`, and `LOWER` for data cleaning and formatting.
* **Conditional Logic:** Implemented `CASE WHEN` for data categorization and `COALESCE` to handle missing values.
* **Relational Joins:** Performed `JOIN` operations to connect entities across multiple tables.
* **Aliasing:** Used `AS` to provide clear and descriptive headers for calculated columns.

## Task Breakdown & Logic:
1. **Data Selection:** Basic filtering by country and age criteria.
2. **Segmentation:** Categorizing customers into 'Young', 'Adult', and 'Senior' groups.
3. **Pattern Matching:** Finding specific records using the `LIKE` operator.
4. **Data Formatting:** Merging columns and changing text case for reporting.
5. **Data Integrity:** Measuring string lengths and providing default values for NULL entries.
6. **Multi-table Analysis:** Joining shippings with customer data to track delivery statuses.
