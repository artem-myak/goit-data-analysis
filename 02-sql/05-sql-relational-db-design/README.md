# Relational Database Design & Data Manipulation

## Project Overview
This project focuses on creating a relational database structure from scratch, establishing table relationships, and performing data manipulation tasks (CRUD operations). The goal was to build a multi-table system for managing employees, departments, and job titles.

## Technical Skills Demonstrated:
* **Database Schema Creation (DDL):** Developed tables with defined data types and Primary/Foreign Key relationships.
* **Data Manipulation (DML):** Executed `INSERT`, `UPDATE`, and `DELETE` operations.
* **Complex Joins:** Used `INNER JOIN` for comprehensive reporting and `LEFT JOIN` to identify gaps in data (departments without employees).
* **Data Integrity:** Ensured referential integrity between tables using Foreign Keys and handled table removal using `CASCADE`.

## Deliverables
* `queries.sql`: Full SQL script including table creation, data manipulation, and join queries.
* `DB-Design-Results.xlsx`: Exported results of the executed queries.
