# Exploratory Data Analysis (EDA) with Pandas

## Project Overview
This project demonstrates the fundamental steps of Exploratory Data Analysis using Python and the Pandas library. The analysis covers data cleaning, handling missing values, statistical computation, and feature engineering based on a fitness tracking dataset.

## Key Analytical Steps:
* **Data Sanitization:** Converting data types (datetime) and inspecting table structures.
* **Missing Data Management:** Calculating null fractions and implementing mean imputation while tracking missingness with flags.
* **Statistical Analysis:** Computing descriptive statistics and identifying anomalies using the **IQR (Interquartile Range)** method.
* **Feature Engineering:** Creating new metrics like `pulse_range` and categorizing records using custom conditional logic.
* **Data Aggregation:** Grouping data to find correlations between workout duration and physical metrics.
