# Data Visualization with Matplotlib and Seaborn

## Project Overview
This project focuses on exploratory data analysis through advanced visualization techniques. Using a real estate dataset, I explored relationships between property characteristics, pricing, and construction years using Python's most powerful plotting libraries.

## Key Visualizations & Techniques:
* **Matplotlib:** Created scatter plots for correlation analysis and line charts using quantile-based pricing groups (`pd.qcut`).
* **Seaborn Statistical Plots:** * Used `scatterplot` with categorical `hue` to differentiate property types.
    * Implemented `boxplot` to analyze price distributions and detect outliers.
    * Used `kdeplot` for density estimation of house areas across top locations.
* **Categorical Analysis:** Applied `countplot` and horizontal `barh` charts with value labels for clear frequency distribution.
* **Styling & Context:** Configured global aesthetics using `sns.set_style` and `sns.set_context` to optimize charts for both notebook exploration and professional presentations.
