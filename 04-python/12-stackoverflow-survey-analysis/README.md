# Stack Overflow Developer Survey Analysis

## Project Overview
This project involves a deep dive into the official Stack Overflow Developer Survey results. Using Python and the Pandas library, I analyzed global trends in programming languages, work experience, remote work adoption, and compensation across different countries and demographics.

## Key Technical Achievements:
* **Large Dataset Handling:** Processed thousands of records and optimized memory usage during data loading.
* **Data Integrity Analysis:** Performed intersection of question sets to identify respondents with 100% completion rates.
* **Statistical Profiling:** Calculated measures of central tendency (Mean, Median, Mode) for work experience and global compensation.
* **Multivariate Filtering:** Identified specific developer segments (e.g., Python developers working remotely in high-income percentiles).
* **Geographical Insights:** Aggregated and compared median vs. average salary levels for specific tech stacks across different countries.

## Skills Demonstrated:
* **Pandas:** GroupBy, Aggregation, Intersection of Sets, Quantile calculations.
* **Data Cleaning:** Complex null handling and string matching for multi-choice questions.
* **Business Intelligence:** Deriving actionable market trends from raw survey data.
